# Inverted Pendulum
This project simulates an inverted pendulum in C++

## Dependencies
- cmake
- make
- gcc/g++
- sfml
- googletest *(optional)*
- sciplot *(optional)*

## Build Instructions
1. Clone this repo.
2. Make a build directory: `mkdir build && cd build`
3. Compile: `cmake .. && make`
4. Run it: `./game`
